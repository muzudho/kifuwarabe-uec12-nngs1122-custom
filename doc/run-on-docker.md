# Run on Docker

```shell
# On Windows.
set COMPOSE_PROJECT_NAME=Kifuwarabe_NNGS_Build1
set NNGS_GIT_USER_NAME=yourname
set NNGS_GIT_USER_EMAIL=yourname@example.com

docker-compose up -d --build
```
